# AI资讯杂志

一个把每日 AI 资讯镜像到本站的静态站点生成器。数据来自 [AI HOT](https://aihot.virxact.com)，每天自动抓取、抽取全文并生成可公网访问的静态杂志。

## 特性
- **五版块日报**：模型发布/更新、产品发布/更新、研究/技术、行业动态、其他，全局连续编号。
- **全文镜像**：服务器端抓取原文 HTML 并用 trafilatura 抽取正文，存到本站「站内统一阅读面板」（点击任意卡片即可在阅读器内查看，无需离开本站）。原文链接与来源署名始终保留，以尊重版权。
- **时间线甘特图**：按美国 / 中国阵营分块展示主要公司（OpenAI、Google、Anthropic、腾讯、阿里…）的模型与产品发布时间线，支持缩放 / 拖动 / 仅看重要事件。
- **永久归档**：自 2026-04-22 起全部历史日报均保留，永不删档。
- **纯静态**：数据在生成时内嵌进 HTML，运行时无需服务器、不调接口、无跨域问题。

## 本地运行
```bash
python generate_archive.py        # 拉取最新日报 + 回填缺失全文 + 重渲染全部页面
```
依赖：`trafilatura`（用于正文抽取）。
产物：`index.html`（杂志首页 + 甘特图）、`ai-daily-YYYY-MM-DD.html`（每日完整日报）、`archive.json`（增量数据清单）。

## 公网部署（GitHub Pages + 每日自动更新）
仓库根目录已含 `.github/workflows/deploy.yml`：
- 每日 **北京时间 08:00** 自动运行（UTC 00:00 cron）；也支持手动触发。
- 自动跑生成器、提交最新数据、部署到 GitHub Pages。
- 首次需开启仓库 **Settings → Pages → Build and deployment → Source: GitHub Actions**。
